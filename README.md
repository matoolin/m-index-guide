# m-index-guide

## 项目简介

`m-index-guide` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 页面的集中存放与版本管理空间。

你可以在此仓库中找到一系列独立的 HTML 文件，每个文件通常对应一个功能或展示页面，适合用于学习、参考、快速测试或本地演示。

## 目录说明

```
m-index-guide/
├── pages/             # 存放所有独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/            # 公共资源文件（CSS、JS、图片等）
├── README.md          # 本文件
└── LICENSE            # 开源许可证
```

- **pages/**：所有可独立运行的 HTML 文件均放置于此目录下，文件名应反映页面用途。
- **assets/**：用于存放页面之间共享的样式、脚本或图片文件，方便维护与复用。
- **LICENSE**：当前仓库使用的许可证文件。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立归档，互不依赖。你可以直接通过浏览器打开 pages 目录下的任意文件进行查看或使用。

若希望将页面部署至静态托管服务，可将整个仓库内容（或仅 pages 目录）上传至支持 HTML 的服务器，即可通过对应路径访问各页面。

## 维护说明

- 新增页面：请在 pages/ 目录下创建新的 HTML 文件，并确保其内部引用资源路径正确。
- 修改页面：请直接编辑对应 HTML 文件，建议在提交时附带简要的修改说明。
- 资源管理：若页面使用公共资源，请将资源文件放入 assets/ 目录，并在 HTML 中通过相对路径引用。
- 版本控制：所有历史修改均通过 Git 记录，建议每次提交只涉及一个页面的改动，以便追踪。

## 贡献

欢迎提交 Issue 或 Pull Request 来改进本仓库。请确保你的贡献不包含任何具体域名或营销内容，保持页面内容的通用性与可读性。
